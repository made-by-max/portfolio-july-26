    The OlySense Report product includes a feature for adding smart text blocks to a medical report. Doctors can use a slash command to add individual findings, standard text blocks, and their own custom text blocks.

    While designing a more complex dropdown menu in Figma, I suggested using a combobox component. My colleagues weren’t familiar with comboboxes, so I built a quick demo with Shadcn components to demonstrate.
